## 2. 代码接入约定

```ts
type ViewMode = 'personal' | 'god'

interface AssetResource {
  id: string
  full_path: string
  type: 'background' | 'panel' | 'button' | 'icon' | 'portrait' | 'role_intro' | 'share_card' | 'bgm' | 'sfx' | 'voice'
  usage: string
  pages: string[]
  modes?: ViewMode[]
}

interface PageAssetBundle {
  id: string
  route: string
  name: string
  mode: ViewMode | 'all'
  components: string[]
  assets: string[]
}
```

```ts
import manifest from '@/assets/werewolf_asset_manifest_v5.json'

const assetMap = Object.fromEntries(manifest.resources.map(item => [item.id, item]))

export function getAsset(id: string) {
  return assetMap[id]?.full_path
}

export function getPageAssets(pageId: string) {
  const page = manifest.pages.find(p => p.id === pageId)
  return page?.assets.map(getAsset).filter(Boolean) ?? []
}
```

## 4. 视角权限与资源规则

### 4.1 个人视角

- 自己：显示真实身份卡、真实角色头像、自己的技能按钮。
- 其它 AI 玩家：默认显示未知身份卡和未知头像。
- 白天公开信息：发言、投票、警长、死亡、放逐、自爆等可以显示。
- 夜晚隐藏信息：狼人击杀目标、女巫用药、预言家查验、猎人状态等不可显示，除非该行为属于玩家自己。
- 结算后：可以解锁全部身份，用 `card_ai_revealed_base` 和真实角色头像渲染。

### 4.2 上帝视角

- 全部玩家真实身份可见。
- 全部夜晚操作可见。
- 全部技能记录、投票、死亡原因、AI 决策可见。
- 使用 `assets/ui/role_cards/god/` 下的真实身份卡资源。

### 4.3 身份介绍卡

身份介绍卡是“规则说明资源”，个人视角也可以查看全部角色说明，但不能显示其它玩家实际身份。

## 9. 角色介绍卡文案

### villager

```text
角色：平民
阵营：好人阵营
类型：民牌
技能：无特殊技能。
操作时机：白天参与发言、警长竞选和放逐投票。
胜利目标：找出并放逐所有狼人。
注意事项：你没有夜晚信息，需要通过发言、投票和逻辑推理判断狼人。
```

### werewolf

```text
角色：狼人
阵营：狼人阵营
类型：狼牌
技能：每晚与其它狼人共同选择一名玩家击杀；白天可选择自爆。
操作时机：夜晚狼人行动阶段；白天发言期间可自爆。
胜利目标：杀光所有神职，或杀光所有平民。
注意事项：需要隐藏身份，伪装成好人，并通过发言和投票误导好人阵营。
```

### seer

```text
角色：预言家
阵营：好人阵营
类型：神牌
技能：每晚可以查验一名玩家，得知其阵营为好人或狼人。
操作时机：夜晚预言家行动阶段。
胜利目标：帮助好人阵营找出所有狼人。
注意事项：查验结果只有自己知道，需要在白天合理传递信息。
```

### witch

```text
角色：女巫
阵营：好人阵营
类型：神牌
技能：拥有一瓶解药和一瓶毒药。每晚只能使用一瓶药，且不可自救。
操作时机：夜晚女巫行动阶段。
胜利目标：帮助好人阵营找出并清除所有狼人。
注意事项：药水数量有限，使用时机非常关键。
```

### hunter

```text
角色：猎人
阵营：好人阵营
类型：神牌
技能：若猎人出局时未被女巫毒杀，可以开枪带走一名玩家。
操作时机：猎人出局结算时。
胜利目标：帮助好人阵营找出并清除所有狼人。
注意事项：开枪目标需要谨慎选择，错误开枪可能帮助狼人。
```

### idiot

```text
角色：白痴
阵营：好人阵营
类型：神牌
技能：若白痴被白天放逐，可以翻牌免于出局，但之后失去投票权。
操作时机：白天被放逐结算时。
胜利目标：帮助好人阵营找出并清除所有狼人。
注意事项：翻牌后仍可参与发言，但不能继续投票。
```

### 实现注意

1. 生成图片时尽量不要让模型直接生成中文 UI 正文，文字应由前端渲染。
2. 面板、按钮、图标建议透明 PNG 或 SVG；背景使用 1920x1080。
3. UI 面板建议按九宫格切图或直接用 CSS 复刻，图片只作为视觉底纹。
4. 所有资源以 JSON manifest 中的 `id` 为稳定引用，不要在代码里硬编码随机文件名。
5. 个人视角必须走 `canShowPlayerRole` 权限判断，禁止直接渲染其它玩家真实身份。

### 角色权限判断示例

```ts
function canShowPlayerRole(params: {
  viewMode: 'personal' | 'god'
  isSelf: boolean
  isGameOver: boolean
  isRoleRevealed: boolean
}) {
  if (params.viewMode === 'god') return true
  if (params.isSelf) return true
  if (params.isGameOver) return true
  if (params.isRoleRevealed) return true
  return false
}
```
